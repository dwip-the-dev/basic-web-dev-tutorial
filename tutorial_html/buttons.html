6) Common Button Attributes

type: defines button behavior.

disabled: disables user interaction.

name and value: useful when button is part of form submission.

autofocus: focuses button automatically when page loads (use carefully).

Tip: Always set type explicitly on buttons to avoid accidental form submission.
