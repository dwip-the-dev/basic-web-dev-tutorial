HTML Lists (Beginner Friendly)

HTML has 3 main list types: unordered list, ordered list, and description list.

1) Unordered List (<ul>)

  • Milk
  • Bread
  • Eggs

Unordered list with another bullet style:

  • HTML
  • CSS
  • JavaScript

2) Ordered List (<ol>)

  1. Open your editor
  2. Create an HTML file
  3. Run it in browser

Ordered list with uppercase letters:

  1. Install tools
  2. Write code
  3. Test code

Ordered list starting from 5:

  1. Step Five
  2. Step Six
  3. Step Seven

Reversed ordered list:

  1. Third
  2. Second
  3. First

3) Description List (<dl>)

HTML
Used to structure web page content.
CSS
Used to style how the page looks.
JavaScript
Used to make the page interactive.

4) Nested Lists (List inside List)

  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python

Tip: Use <ul> for bullet points, <ol> for steps/order, and <dl> for term + meaning.
