Favicon Tutorial (Beginner Friendly)

This page uses assets/favicon.ico as the active favicon. Open this file in a browser tab and you should see the icon on the tab.

Quick Checklist

1. Keep at least one .ico file for compatibility.

2. Add PNG/SVG versions for modern quality and flexibility.

3. Add Apple touch icon and manifest if building a web app/PWA.

4. If icon changes do not appear, use cache-busting like ?v=2.
